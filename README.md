# key-chain-design

In this project i used Onshape to design a keychain with the daredevil logo.

To design this i used multiple sketches since i needed multiple extruded shapes.

After designing the shape i realised that the design was too big to be a keychain i used the transform feature to make the whole part smaller without redrawing the sketch.

I used 4mm for the keyway and 2mm for the outside parts (red ones) and 1mm for the inside part (black one).
